## 多种布局方式

### 双栏布局

```html
//BFC
<style>
  .left {
    float: left;
    width: 200px;
    height: 200px;
    background-color: red;
  }
  .right {
    overflow: hidden;
    background-color: blue;
    height: 500px;
  }
</style>

<div>
  <div class="left"></div>
  <div class="right"></div>
</div>
```

```html
//flex
<style>
  .container {
    display: flex;
  }
  .left {
    width: 200px;
    height: 200px;
    background-color: red;
  }
  .right {
    flex: 1;
    background-color: blue;
    height: 500px;
  }
</style>

<div class="container">
  <div class="left"></div>
  <div class="right"></div>
</div>
```


```html
//grid
<style>
  .container {
    display: grid;
  }
  .left {
    width: 200px;
    height: 200px;
    background-color: red;
  }
  .right {
    flex: 1;
    background-color: blue;
    height: 500px;
  }
</style>

<div class="container">
  <div class="left"></div>
  <div class="right"></div>
</div>
```
